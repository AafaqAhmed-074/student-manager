# 🎓 Student Manager

A simple full-stack Student Management system built using:

- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express.js
- Database: MongoDB (Mongoose)
- API: RESTful APIs

---

## 🚀 Features

- Add student
- View students
- Store data in MongoDB
- REST API integration
- Simple UI


---

## 🛠️ Tech Stack

- Node.js
- Express.js
- MongoDB Atlas
- Mongoose
- HTML/CSS/JavaScript

## ⚙️ Installation & Setup

### 1. Clone the repository

```bash
git clone https://github.com/your-username/student-manager.git



**Install dependencies**

npm install



**Setup MongoDB**

Create a free database:

👉 https://www.mongodb.com/atlas/database

Then replace your connection string inside:
server/db.js

**Run the project**
npm start


**Server will run on:**
http://localhost:5000
